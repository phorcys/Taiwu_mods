### 使用方法

ctrl+F10的设置界面设置需要加速的建筑，每一个对应于一个坐标(x, y)，这个坐标是相对于13x13的家里中央"太吾村"而言的，x、y方向和通常的直角坐标一样，x左负右正，y上正下负，太吾村算(0, 0)，所以x、y绝对值最大是6。举例：(5, -6)表示以太吾村出发向右5格，下6格。
