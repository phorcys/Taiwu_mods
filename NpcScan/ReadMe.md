# NPC查找器（NPCScan）

## 使用说明

在游戏中通过Ctrl+F12唤出NPC搜索窗口

*by：Saharaqie*

B站ID：撒哈拉来的企鹅

终极目标
在游戏里面增加一个功能，通过某种方式获取各种NPC的信息
增加游戏内置排行榜

更新计划
检索隐藏的性取向
