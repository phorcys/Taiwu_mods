# 简介
可自定义配置的键盘快捷键。

快捷键大致分为以下几种：

1. 各种对话框确认、取消快捷键；
1. 大地图界面各按钮快捷键；
1. 战斗界面功法施展快捷键；
1. 对话界面快捷键。

详情请在 mod 配置界面查看。
