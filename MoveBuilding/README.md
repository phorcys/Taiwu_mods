# MoveBuilding
太吾绘卷的MOD插件，太吾村搬移建筑物

# 使用注意
本功能现在仍是测试阶段，

若担心搬移建筑后，储存載入发生问题，建议使用前先备份存档

# 使用方法
在太吾村，用滑鼠右键，拖动建筑物，放在空地

不允许搬动位于太吾村中心的太吾村

不允许放置建筑物在不能新增的位置

不要移动正在升级/拆除的建筑

移动收益建筑前，请把收益拿走。反正你移了就没了。
